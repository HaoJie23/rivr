Class VoiceXmlDialogueContext

  • java.lang.Object
    • com.nuecho.rivr.voicexml.dialogue.VoiceXmlDialogueContext
  • All Implemented Interfaces:
    DialogueContext<VoiceXmlInputTurn,VoiceXmlOutputTurn>

    public final class VoiceXmlDialogueContext
extends java.lang.Object
implements DialogueContext<VoiceXmlInputTurn,VoiceXmlOutputTurn>
    VoiceXML specialization of DialogueContext. In addition to properties provided by a DialogueContext (like dialogueChannel, dialogueId, and logger), the VoiceXmlDialogueContext exposes:
    • servlet information used to build URI of some resources.:
      • servletPath
      • contextPath
    • VoiceXML properties (for each generated document)
    • VoiceXML default language (for each generated document)
    • turnIndex
